# 瞬発思考

「ゼロ秒思考」メソッドをWeb版にアレンジしたアプリ。原作の1分間を20秒に短縮し、より手軽に思考トレーニングを行える。

## コンセプト

- 原作「ゼロ秒思考」（赤羽雄二氏）へのリスペクト
- 1分 → 20秒にアレンジし、より気軽に継続できる形に
- フロントエンド完結で動作（バックエンド不要）

## ターゲットユーザー

- ビジネスパーソン
- 仕事の生産性向上・思考整理をしたい人

## 基本フロー

1. 繰り返し回数を決める（デフォルト10回）
2. **書き出しモード** - テーマについて思いついたことを書く
3. **読み返しモード** - 20秒経過後、書いた内容を確認（追記も可能）
4. 「次へ」で再び書き出しモードへ
5. 10回繰り返す
6. **出力モード** - 全ての内容をコピペ可能な形式で表示。AIエージェントへの貼り付けを促す

## 主な特徴

- 残り時間は常に大きく表示
- スマホでも書きやすいUI
- Google AdSense広告を常時表示

## デザイン

- **コンセプト**: ノートのような黄色い優しい色味
- **雰囲気**: 知的好奇心をくすぐるノート風デザイン

## 技術スタック

- **フレームワーク**: React + TypeScript
- **データ保存**: localStorage（ブラウザ内保存）
- **ホスティング**: GitHub Pages
- **広告**: Google AdSense

## MVP（最小構成）

- タイマー機能（20秒カウントダウン）
- テキスト入力機能
- 結果出力機能（コピー可能）

## ゴール

1. **短期**: 1円でも稼げる状態にする
2. **中期**: 宣伝し、自分で3週間継続使用する
3. **長期**: モバイルアプリ化 → サブスクリプション化

コード品質チェック用のカスタムスラッシュコマンドを用意しています。

| コマンド | 説明 |
|---------|------|

#### 使用例

```bash

/review:atomic src/components/  # 特定パスのみ

# テストレビュー
/review:test                # 全体チェック
```
