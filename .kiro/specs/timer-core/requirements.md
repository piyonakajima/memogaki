# Requirements Document

## Project Description (Input)

「瞬発思考」アプリのタイマーコア機能。

### 対象機能
- 20秒カウントダウンタイマー
- 開始/停止/リセット機能
- 残り時間の大きな表示
- タイマー終了時のコールバック

### プロジェクト概要（README.mdより）
「ゼロ秒思考」メソッドをWeb版にアレンジしたアプリ。原作の1分間を20秒に短縮し、より手軽に思考トレーニングを行える。

### 関連仕様
- 残り時間は常に大きく表示（視認性重視）
- タイマー終了時に次のモード（読み返しモード）へ遷移するためのコールバックが必要
